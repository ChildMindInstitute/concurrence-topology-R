This folder contains the Concurrence Topology software written by S.P. Ellis. 
(This is contained in the R data file "ConcurrenceTopology.RData". 
This folder also contains documentation (in the "Documentation" folder), 
and examples (in "Examples"). Two files "LICENSE.txt" and "NOTICE.txt" 
describe licensing.

To learn how to use the software, read the file 
"Documentation/ConcurTopolSoftwareHowTo.txt".
