(3/1/'13)

This directory contains supplementary material for the paper "Describing high-order statistical dependence using 'Concurrence Topology', with application to functional MRI brain data" by Ellis and Klein (2013).

The file "SomeBackground.pdf" covers some topics concerning concurrence topology not covered in the paper. There are also three subdirectories. "fMRI_data" contains the data used for the paper. These include the "demographic" data (including diagnosis) and BOLD data. "ConcurrenceTopology_software" contains the software and documentation. 

The folder "Some_Results" contains some findings concerning the data.
